== Description ==

Create easily circle contents for your website or blog. [Here some examples](http://www.info-d-74.com/easy-circle-contents-plugin-wordpress-demos/)

And you can check the [Pro version](https://www.info-d-74.com/produit/easy-circle-contents-pro-plugin-wordpress/) that add many options.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to the Easy circle contents
